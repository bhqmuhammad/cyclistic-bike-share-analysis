1. The Ask

To drive future growth, we need to convert more casual riders into annual members.

Our Primary Question:

How do annual members and casual riders use Cyclistic bikes differently?

2. Prepare

We used Cyclistic's historical trip data from Q1 2019 and Q1 2020.

Key Preparation Steps:

  • Combined two datasets into a single source of truth.
  • Calculated `ride_length` and `day_of_week` for each trip.
  • Cleaned the data by removing duplicates and null values to ensure accuracy.

3. Process

The cleaned data was analyzed using Python with the Pandas library. Initial data exploration helped identify key patterns, which were then prepared for visualization.

The final visualizations for this presentation were created using Chart.js to provide clear, interactive insights.

4. Analyze

Our analysis revealed two major differences in usage patterns:

  • Ride Duration: Casual riders take significantly longer trips on average.
  • Weekly Usage: Annual members ride consistently throughout the week (likely for commuting), while casual riders are most active on weekends.

6. Act: Recommendations

Based on our findings, we propose a three-pronged marketing strategy:

  1. Targeted Weekend Promotions: Create a "Weekend Warrior" membership to appeal to casual riders' peak usage time.
  2. Highlight Commuter Benefits: Run campaigns showing the cost savings of an annual membership for daily work commutes.
  3. Introduce Tiered Memberships: Offer a lower-cost, limited-ride membership as an entry point to convert hesitant casual riders.
